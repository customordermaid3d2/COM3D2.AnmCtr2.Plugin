# 포함된 내용

- 기어 메뉴 추가  
- 설정파일에 값 저장  
- 단축키 설정  
- GUI 기본 작성. 윈도우 리사이즈시 밖으로 나가버리는거 방지 포함  
- 코루틴  
- 하모니  
- OnSceneLoaded 기본 사용법.  scene.buildIndex는 쓰지 말자 제발.
